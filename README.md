# MaiTincMerch! — Vue 3 + Vite

Proyecto convertido desde React por CDN/Babel a Vue 3 con Vite.

## Instalar

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build producción

```bash
npm run build
```

## Preview del build

```bash
npm run preview
```

## Estructura

- `src/App.vue`: composición principal de la landing.
- `src/components/`: secciones convertidas a componentes Vue.
- `src/styles.css`: estilos originales adaptados para Vite.
- `public/assets/`: imágenes y recursos estáticos.
